## ❓ Common Questions

### Q: Where's my data stored?
**A**: In `finance.db` (SQLite database file). It's local on your computer.

### Q: Can I backup my data?
**A**: Yes! Settings → App Settings → Backup to export as JSON.

### Q: How do I reset everything?
**A**: Settings → App Settings → Reset to Defaults (careful - this wipes data!)

### Q: How do I add real OpenAI API?
**A**: See QUICK_START.md or README_COMPLETE.md for integration steps.

### Q: Is my data secure?
**A**: Yes! Passwords are hashed, all data is local, SQL injection prevented.

### Q: Can I use this on multiple computers?
**A**: Currently no (v1.0). Cloud sync coming in v1.1+.

## 🐛 Troubleshooting

### App won't start
1. Check Python version: `python --version` (need 3.8+)
2. Install dependencies: `pip install -r requirements_complete.txt`
3. Check for errors in console

### Database errors
1. Delete `finance.db`
2. Run app again to recreate
3. Demo data will reload

### UI looks strange
1. Check screen resolution (need 1024x768+)
2. Restart application
3. Try resizing window

### Performance slow
1. Check available RAM (need 4GB+)
2. Close other applications
3. Restart computer
